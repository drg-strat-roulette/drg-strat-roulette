We'd love to get your feedback!

There are a number of ways you can give this project feedback. DRG Strategy Roulette is hosted in a public GitHub repository. On there, you can report a bug, or participate in discussions. Discussions provide a forum to post whatever you want. You can ask questions, share ideas/suggestions, or simply share your experiences!

Our GitHub is the preferred way to provide feedback, but if you'd like to provide anonymous feedback and/or don't want to create a GitHub account, you can also submit feedback anonymously through our Google Form.
